# radioCast

## a Podcast application where you can play any Podcast on your phone at any time.

### tools used to build this app:

    - React Native.
    - typescript.
    - SQLite.
    - React Context.
    - React Navigation.
    - Detox - Jest.
    - react-native-track-player
    - react-native-size-matters
    - react-native-svg
